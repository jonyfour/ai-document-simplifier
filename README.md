# AI Document Simplifier

AI-powered Google Sheets automation that simplifies complex documents into plain English,
extracts key obligations/risks, and generates keywords.

## Features
- Input technical/legal text
- Outputs plain English summary
- Extracts key risks/obligations
- Generates keywords
- Stores everything automatically with date

## Tech Stack
- Google Apps Script
- Groq LLM API
- Google Sheets
